## Setup
### npm
1. install nodebrew
  - `brew install nodebrew`
1. install npm
  - `nodebrew install v14.14.0`
  - `nodebrew use v14.14.0`
1. set path to npm
  - `echo 'export PATH=$PATH:$HOME/.nodebrew/current/bin' >> ~/.bash_profile`

### Anguler (first time)
1. `npm install @angular/cli`
1. `/ng new client`
  - N, stylus
  
  `npm install stylus`

## run
`./node_modules/@angular/cli/bin/ng serve --open`
